# UNC Charlotte Super Fab Lab Senior Design Project
## OLSK Small 3D Printer V2.1
The repo includes an `index.html` file (so it can render a web page), two GitHub Actions workflows, and a CSS stylesheet dependency.
![Image of Electronics Bay, replace soon with image of finished printer.](Images/IMG_0260.jpeg)

## Introduction
The OLSK Small 3D Printer V2.1 is a Senior Design project under the Senior Design Program at the University of North Carolina at Charlotte. The team was tasked to research, present, and prototype enhancements during the two semester long program.
